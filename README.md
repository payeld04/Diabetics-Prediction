# Diabetics-Prediction using ML and Python


This project focuses on predicting the likelihood of diabetes in individuals using machine learning algorithms and Python. By leveraging healthcare datasets, the model aims to assist in early detection and risk assessment, which can be crucial for preventive care and medical decision-making.

### Objectives:
- Build a predictive model for diabetes diagnosis.
- Explore different machine learning algorithms for accuracy and efficiency.
- Provide a reproducible workflow for healthcare-related predictive analytics

### Dataset:
The project uses the Pima Indians Diabetes Dataset, a widely recognized dataset in medical machine learning research.
It contains features such as:
- Number of pregnancies
- Glucose concentration
- Blood pressure
- Skin thickness
- Insulin levels
- BMI (Body Mass Index)
- Diabetes pedigree function
- Age
- Outcome (1 = diabetic, 0 = non-diabetic)

### Technologies Used:
- Python
- Libraries: numpy, pandas, scikit-learn

### Machine Learning Models:
- Support Vector Machine(SVM)
- Linear Regression

### Workflow:
- Data Preprocessing
- Exploratory Data Analysis (EDA)
- Model Training & Evaluation
- Model Comparison

### How to Run:
- Clone repository or,
- Open the notebook directly in Google Colab:
```bash
https://colab.research.google.com/github/your-username/diabetes-prediction/blob/main/Diabetes_Prediction.ipynb
```

### Results:
- Models are evaluated using accuracy.
- Linear Regression and SVM classifier model generally perform well.
- Feature importance plots highlight key predictors of diabetes.
